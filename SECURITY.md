# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing **team@wordsworth.ai** with the subject line `[SECURITY] wwai-agent-orchestration`.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce (proof of concept if available)
- Any suggested fixes or mitigations

We will acknowledge receipt within **48 hours** and aim to provide a resolution or mitigation plan within **7 days** for critical issues.

## Scope

Security issues in scope include:
- Credential or secret exposure in the codebase or published artifacts
- Dependency vulnerabilities with a clear impact path
- Code execution vulnerabilities in any node or workflow component

## Disclosure

This project is provided as-is. Once a fix is released, we will publish a summary of the vulnerability in the GitHub release notes.
